CSS 笔记

CSS syntax and selectors

CSS syntax

p {color: red; text-align: center;}

CSS Selectors
  1. element selector
    p {color: red; text-align: center;}
  2. id selector
    #para1/p#para1 {color: red; text-align: center;}
  3. class selector
    .center/p.center {color: red; text-align: center;}
    HTML elements can also refer to more than one class. But not id
  4. Grouping selector
    h1, h2, p {text-align: center;color: red;}

CSS Borders

  1. Border
    p {border: 5px solid red;}

    style 必须指定，其他可以默认

  2. Border-style/color/width/radius
    p {Border-style/...: solid dotted dashed;}
    与每个边的对应关系（2-4value）：
    上下／右左；上／下／右左；上／右／下／左（顺时针）

    • style: dotted/dashed/solid/double/groove/ridge/inset/outset/none/hidden/
  3. Border-left/…
    • 简化时，同Border一样,必须指定style。
    • Border-left/…-style/… 这是最具体的指定。
property value
border-style/color/width/radius
border-top/right/bottom/left-style/color/width
border Shorthand(四条边一样)
border-top/right/bottom/left Shorthand(每条边单独指定)

CSS Margins

  1. margin-top/right/bottom/left
  2. margin (Shorthand) : 同border，与每个方位的对应关系。
  3. 几个特性：
    • inherit: margin:inherit;
    • auto value: margin:auto;
    • collapse: 上行bottom与下行top 的margin会collapse，只取最大的那个，不是相加。（左右的不collapse）

CSS Padding

  1. padding-top/right/bottom/left
  2. padding(Shorthand): 对应关系同上。
  3. inherit: 同上。（padding没有auto和collapse）

CSS Height/Width

  1. height/width
    • Note: The height and width properties do not include padding, borders, or margins; they set the height/width of the area inside the padding, border, and margin of the element!
  2. max-width(min height)
    • max-width适应浏览器窗口大小；width固定大小。
    • Note: The value of the max-width property overrides width.

CSS Box Model

  1. The CSS Box Model
    • Content - The content of the box, where text and images appear
    • Padding - Clears an area around the content. The padding is transparent
    • Border - A border that goes around the padding and content
    • Margin - Clears an area outside the border. The margin is transparent
  2. Width and Height of an Element
    • Important: When you set the width and height properties of an element with CSS, you just set the width and height of the content area. To calculate the full size of an element, you must also add padding, borders and margins.
    • The total width : 四样element相加
    • The total height : 四样element相加

CSS Layout

css Display

  1. default display value
    • block-level elements : <div> <h1> <p> <header> <form> <footer> <section>
    • inline elements : <span> <a> <img>
  2. override the default display value
    • display:inline
    • display:block
  3. hide element
    • display:none : not take up space
    • visibility:hidden : take up the same space as before

css max-width

  1. width : 文本不随窗口大小变化，不方便小屏设备查看
  2. max-width : 文本随窗口大小折叠，方便小屏设备查看

css Position

Position Property
  1. position:static : 静态的（静止），不因top bottom left等属性而发生位移。
  2. position:relative : relative to its normal position .
  3. position:fixed : relative to the browser window.
  4. position:absolute : relative to the HTML page, or relative to the nearest positioned ancestor.
Overlapping Elements
  1. z-index : 越小（可为负数）越底下
  2. default : 默认 absolute 在最上层；后出现的在上层。
  3. Positioning Text in an image : 用 position:absolute relative to the nearest positioned ancestor.

float and clear

float and clear
property description
float:right/left Specifies whether or not an element should float
clear:right/left 不允许哪边float

Inline-block

display: inline-block 效果等同于多个box时的 float: left 和 clear: left 的结合 ，可以更简单地表示相同的效果。

但是两种表达方式本质上是不同的，float是漂浮在XXX之上，而display是指定element以什么方式表现出来。

css Align

  1. center Align Elements : margin: auto;
  2. Center Align Text : text-align: center
  3. Center an Image : margin: auto; display: block; image默认是inline的element，所以要定制成block。
  4. Left/Right Aligh :
    • using position : position: absolute; right: 0px;
    • using float : float: right;
  5. Center Vertically :
    • using padding : padding: 50px 0;
    • using line-height : line-height 和 height 的值设成一样的； line-height 是text的高度。
    • Using position & transform : transform: translate(-50%, -50%);

css Pseudo-classes

A pseudo-class is used to define a special state of an element.

Syntax
  1. Anchor Pseudo-classes:
    a:link/visited/hover/active
  2. Pseudo-classes and css classes:
    a.highlight:hover 类和伪类的组合， hightlight是类名
  3. Hover on <div>
    div:hover {background-color: blue;}
  4. Tooltip Hover : 是combinator和hover的结合

注：就是先隐藏 p element，然后用 div:hover 显示。

  1. ：first-child Pseudo-classes
    • p:first-child {color: blue;}

  2. :lang Pseudo-classes

css Pseudo-element

syntax

selector::pseudo-element {property:value;}

selector example description
::after p::after , insert sth. after each <p> element
::before p::before , insert sth. before each <p> element
::first-letter p::first-letter , selects the first letter of each <p> element
::first-line p::first-line , selects the first line of each <p> element
::selection p::selection , selects the portion of an element that is selected by a user
Pseudo-element and css classes

e.g. p.XYZ::first-letter{color:...} XYZ是class。

Multiple Pseudo-elements

p::first-letter and p::first-line 二者的作用域有重叠，（注：后者不更改前者已经生效的）
