CSS syntax and selectors

CSS syntax

p {color: red; text-align: center;}

CSS Selectors
  1. element selector
    p {color: red; text-align: center;}
  2. id selector
    #para1/p#para1 {color: red; text-align: center;}
  3. class selector
    .center/p.center {color: red; text-align: center;}
    HTML elements can also refer to more than one class. But not id
  4. Grouping selector
    h1, h2, p {text-align: center;color: red;}

CSS Backgrounds

  1. background color
  2. background image
    • body {background-image: url("paper.gif");}
    • repeat
      • repeat horizontally or vertically :
        background-repeat:repeat-x/y
      • no repeat : no-repeat
    • position
      • set position : background-position: right/left top
      • fixed position : background-attachment:fixed
  3. Shorthand property
    body {background: #ffffff url("img_tree.png") no-repeat right top;}

CSS Borders

  1. Border
    p {border: 5px solid red;}

    style 必须指定，其他可以默认

  2. Border-style/color/width/radius
    p {Border-style/...: solid dotted dashed;}
    与每个边的对应关系（2-4value）：
    上下／左右；上／下／左右；上／左／下／右

    • style: dotted/dashed/solid/double/groove/ridge/inset/outset/none/hidden/
  3. Border-left/…
    • 简化时，同Border一样,必须指定style。
    • Border-left/…-style/… 这是最具体的指定。
