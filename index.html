HTML Tag Reference

Tag Description

<head> syntax：

<head>

HTML paragraphs：

Tag Description
<p> Defines a paragraphs
<br> Inserts a single line break
<pre> Defines pre-formatted text
备注：

With HTML, you cannot change the output by adding extra spaces or extra lines in your HTML code.

The browser will remove extra spaces and extra lines when the page is displayed.

Any number of spaces, and any number of new lines, count as only one space.

HTML colors

三种表示方法：
use way syntax
Color Names style="color:red"
RGB style="color:rgb(255,0,255)"
Hexadecimal Colors style="color:#FF00FF"
备注：
  1. 可以是color或background－color
  2. 16进制，每两位分别与红、绿、蓝对应，区间0-255

HTML Styles - CSS

  1. inline : <h1 style="color:blue;"> text </h1>
  2. 初始规范型 : 在<head>里.
    • <style>element : 名称 {属性:值;} 相当于定义函数：名称＋{} ；名称可以是body p h1 ，属性包括：color background-color font-family font-size border padding margin 等。The CSS Box Model 如下：
      p { border: 1px solid black;
      padding: 10px;
      margin: 30px; }
    • link : 链接外部css文件，放在<head>里，但不用<style>。<link rel="stylesheet" href="styles.css">
  3. 引用型：在<head>里 <style>下定义id和class.
    • id : #p01 {color:blue;} body里用id=”p01”来引用。
    • class : p.error {} body里用class=”error”来引用。

HTML links

syntax:

<a href="url">link text</a>

links 方式:
  1. an absolute URL (A full web address)
  2. A local link (the same web site) relative URL (without http://www).
  3. Create a Bookmark: link use href="#value"; id use id="value"
links 样式:
  1. links colors: 在<head> <style> 里用a:link a:visited a:hover a:active 分别规范不同操作的效果
  2. HTML Links - The target Attribute
    1. _blank 在新窗口打开
    2. _self 在同一个窗口打开（默认）
    3. _parent Opens the linked document in the parent frame
    4. _top Opens the linked document in the full body of the window
    5. framename Opens the linked document in a named frame
  1. Image as Link:<a href="url">link text</a>里的link text 换成 <img src... >

HTML Images

syntax:

<img src="url" alt="some_text">

  1. Image Size width height
    • inline style : <img src="url" alt="some_text" style="width height"> 推荐这种，尺寸不受CSS影响。
    • inline 非 style : <img src="url" alt="some_text" wwidth="" height=""> 这种情况会受css影响
  2. url来源： 同一文件夹、不同文件夹、其他服务器。
  3. Image Floating: <img src="url" alt="some_text" style="float:right/left">
  4. Image Maps:
    <img src="url" alt="some_text" usemap="#name">
    <map name=" "> <area ... > </map>

HTML Tables

  • Use the HTML <table> element to define a table
  • Use the HTML <tr> element to define a table row
  • Use the HTML <td> element to define a table data
  • Use the HTML <th> element to define a table heading
  • Use the HTML <caption> element to define a table caption
  • Use the CSS border property to define a border
  • Use the CSS border-collapse property to collapse cell borders
  • Use the CSS padding property to add padding to cells
  • Use the CSS text-align property to align cell text
  • Use the CSS border-spacing property to set the spacing between cells
  • Use the colspan attribute to make a cell span many columns
  • Use the rowspan attribute to make a cell span many rows
  • Use the id attribute to uniquely define one table (同css的引用型)
